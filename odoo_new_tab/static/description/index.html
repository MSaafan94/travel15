View Records In New Tab

This module allows user to open and view records of list in a new tab.

View Records In New Tab provides you the functionality to view each and every record in a new tab. It integrates a new tab icon beside every record thus, making it easier for a user to review records without running Odoo every time.

Information

User Guide
https://webkul.com/blog/odoo-view-record-in-new-tab/

Features:

  • A new button will be added in all records in all list views.
  • On clicking this button the corresponding record will be open in New Tab.
  • A record could be anything like Invoice details, Product details, Sale Order, Purchase Order, etc.
  • This Module will help you to compare different records at the same time.
